WarexAI: Data Insights

  • Client Personal Project
  • Category Web Development
  • Date May, 2023

For the WarexAI project, I embarked on creating a web platform aimed at helping small businesses analyze their statistics and gain actionable insights. This personal project was developed using a combination of Python for backend logic, TypeScript for front-end functionality, along with HTML and Bootstrap for the design and layout. The core feature of WarexAI is its ability to allow users to upload Excel sheets, select a target column for analysis, and then generate detailed charts. Beyond visualization, the platform leverages OpenAI to provide tailored suggestions on improving outcomes related to the selected target column, offering users a unique blend of data analysis and AI-driven recommendations.

Key highlights of the WarexAI project include: Development using Python, TypeScript, HTML, and Bootstrap, ensuring a robust and responsive user experience. An intuitive interface that simplifies the process of uploading Excel sheets and selecting target columns for analysis. Advanced chart generation for visual data analysis, making complex information easily understandable. Integration with OpenAI to offer smart suggestions for business improvement, based on the analyzed data. This project not only showcases my technical skills across a range of tools and languages but also demonstrates my ability to innovate solutions that address real-world business challenges.
