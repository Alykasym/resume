  • Client Lebap Nakliye

  • Date August, 2022

While working at Lebap Nakliye, a logistics and transportation company, I was tasked with developing a revamped version of their existing website. Recognizing the importance of a modern and responsive design in enhancing user experience and brand image, I embarked on designing a new user interface (UI) from the ground up. Utilizing HTML and JavaScript for the website's structure and functionality, paired with Bootstrap for styling, I ensured that the new design was not only aesthetically pleasing but also mobile-friendly and accessible across various devices.

Key aspects of the website redevelopment included:

- A clean, intuitive UI design that simplifies user navigation, making it easier for visitors to find the information they need, from services offered to contact details.
- Implementation of Bootstrap to achieve a responsive layout, ensuring the website's optimal performance on both desktop and mobile platforms.
- Enhanced functionality through JavaScript, improving user interaction with dynamic content and real-time updates without needing to refresh the page.

This project not only improved the online presence of Lebap Nakliye but also contributed to a better customer experience, reflecting the company's commitment to professionalism and quality service. Through this redevelopment, I was able to apply my web development skills in a real-world scenario, delivering a product that meets the modern demands of digital engagement for businesses.
